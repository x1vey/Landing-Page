How to Customize This Page

  • To change any picture: Find background: url('...') in the HTML and replace the URL with your image link.
  • To change the copy: Just edit the text between the tags in each section (e.g., <div>About Me</div>).
  • To change the Book a Call link: Replace the Calendly URL in href="https://calendly.com/your-scheduling-link" with your booking page.
  • To toggle the Special Offer button: At the top of the HTML, set SHOW_OFFER = true or false in the script.
  • To add more testimonials: Copy and paste one of the .testimonial-card blocks inside .testimonials-list.
  • Delete this green instructions section when you’re ready to go live!
